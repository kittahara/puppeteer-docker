## how to use
1. docker up
> docker-compose build
> docker-compose up

2. login node container 
> docker exec -it  puppeteer_example /bin/bash

3. execute app.js
> root@xxxxxxxxxx:/# cd /usr/src/app/script/
> root@xxxxxxxxxx:/usr/src/app/script# node app.js
